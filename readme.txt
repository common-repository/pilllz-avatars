A plugin to replace your gravatars with pilllz avatars.
  
== Description ==

Pilllz is a [free avatar generator](https://pilllz.com). This plugin lets you replace the random Gravatar Avatars that sometimes don't fit with your website.  

  
It's still in beta so feel free to send us some requests if you wish to. You should also make sure you always have the latest version installed.
  
== Installation ==
  
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
1. Activate Pilllz by clicking the checkbox under "Settings" > "Pilllz Avatar Generator"
1. If you obtained special credentials from contact@meepha.com, use them in the form

== Frequently Asked Questions ==
  
= How do I use this plugin? =
  
All you need to do is to activate Pilllz Avatars by clicking the checkbox under "Settings" > "Pilllz Avatars"

= How to uninstall the plugin? =
  
Simply deactivate and delete the plugin.
